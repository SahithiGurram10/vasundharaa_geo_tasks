# React Live Search App

A simple React application that implements a live search feature. As the user types in the search input, the list of items updates in real time based on the entered query.

### 1.Brief of Project
This project demonstrates:

- React functional components
- State management using useState
- Performance optimization using useMemo
- Component-based folder structure
- Real-time filtering (live search)
---

## 2. Features
- Set countdown time in seconds (default: 10 seconds)
- Accepts only positive integers
- Start, Pause, Resume, and Reset controls
- Millisecond-level countdown display
- Displays timer status: Idle, Running, Paused, Completed
- Automatically stops at 0 and shows “Time’s up!”
- Prevents multiple timers from running simultaneously
- Timer state persists across page refresh using localStorage

---

## 3. Tech Stack
- React (Functional Components )
