# Advanced Countdown Timer System

## 1.Brief of Project
This project is an Advanced Countdown Timer built using React.  
It allows users to configure a timer, control its execution, and ensures the timer state persists even after a page refresh.

The application is designed with clean component structure, readable code, and follows best practices expected in a React internship assignment.

---

## 2. Features
- Set countdown time in seconds (default: 10 seconds)
- Accepts only positive integers
- Start, Pause, Resume, and Reset controls
- Millisecond-level countdown display
- Displays timer status: Idle, Running, Paused, Completed
- Automatically stops at 0 and shows “Time’s up!”
- Prevents multiple timers from running simultaneously
- Timer state persists across page refresh using localStorage

---

## 3. Tech Stack
- React (Functional Components & Hooks)
- JavaScript (ES6+)
- HTML5
