# React Developer Intern Assignment

This repository contains the implementation of the React Developer Intern assignment provided by Vasundharaa Geo Technologies. The project covers multiple React concepts such as component-based architecture, state management, form handling, localStorage usage, and dynamic UI updates.

---

## Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- LocalStorage

---

## Steps to Run the Project Locally

### Prerequisites
- Node.js version: v16 or above
- npm version: v8 or above

### Installation Steps
1. Clone the repository:
   `
   git clone <https://github.com/SahithiGurram10/vasundharaa_geo_tasks>
  `
2. Navigate to the project directory:
`
cd <vasundharaa_geo_tasks>
`
3. Install dependencies:
`
npm install
`
## Start Command
`
 npm start
`

## The application will run on:
`
http://localhost:3000
`

## Assignment Tasks Implemented

### Task 1: Todo Application
- Add new tasks using an input form
- Display tasks using component mapping
- Delete tasks from the list
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Set priority for tasks
- Persist tasks using localStorage

### Task 2: Form Handling & Password Toggle
- Form fields: Name, Email, ID, Password
- Prevent page reload on form submission
- Display submitted data below the form
- Show / Hide password functionality
- Basic validation for all fields
- Email format validation
- Inline validation error messages
- Clear form after successful submission

### Task 3: Dynamic Progress Bar
- Multiple numeric input fields (0–100)
- Single main progress bar with sub-bars
- Dynamic progress updates
- Input value validation
- Animated progress bar fill
- Color changes based on percentage

### Task 4: Advanced Countdown Timer
- Configurable timer with default value
- Start, Pause, Resume, and Reset functionality
- Dynamic button enable/disable
- Display timer status (Running / Paused / Completed)
- Millisecond-level countdown display
- Persistent timer state using localStorage
- “Time’s up!” message on completion

### Task 5: Live Search with Highlighting
- Case-insensitive search functionality
- Real-time filtering of results
- Highlight matched text (including multiple occurrences)
- Display count of matching results
- Show “No matches found” when applicable
