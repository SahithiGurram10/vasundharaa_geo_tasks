# React Developer Intern Assignment

This repository contains the implementation of the React Developer Intern assignment provided by Vasundharaa Geo Technologies. The project covers multiple React concepts such as component-based architecture, state management, form handling, localStorage usage, and dynamic UI updates.

---

## Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- LocalStorage

---

## Steps to Run the Project Locally

### Prerequisites
- Node.js version: v16 or above
- npm version: v8 or above

### Installation Steps
1. Clone the repository:
   ```bash
   git clone <your-github-repository-link>
  ```
2. Navigate to the project directory:
```
cd <project-folder-name>
```
3. Install dependencies:
```
npm install
``

## Start Command
```
npm start

```

## The application will run on:
```
http://localhost:3000
```
